# Crypto Trader Skill

This skill provides expertise in cryptocurrency trading analysis, strategy development, and backtesting using the tools in this project.

## Market Data Analysis

Use `BinanceClient` from `src.crypto_analysis.data.binance` to fetch market data.

```python
from src.crypto_analysis.data.binance import create_client

client = create_client()
df = client.fetch_ohlcv("BTCUSDT", interval="1h", limit=100)
```

## Technical Indicators

Implement indicators by inheriting from `Indicator` in `src.crypto_analysis.signals.base`.

```python
from src.crypto_analysis.signals.base import Indicator
import pandas as pd

class MyIndicator(Indicator):
    name = "my_indicator"
    required_columns = ["close"]
    
    def calculate(self, data: pd.DataFrame) -> pd.Series:
        # Implementation logic
        return data["close"].rolling(window=14).mean()
```

## Signal Generation

Signals are generated by `SignalGenerator` classes. They produce `Signal` objects containing symbol, type (ENTRY_LONG, EXIT_LONG, etc.), timestamp, and confidence.

Common generators:
- `TechnicalGenerator`: Uses technical indicators.
- `MLGenerator`: Uses machine learning models.
- `StatisticalGenerator`: Uses statistical methods.

## Strategy Development

Combine multiple signal generators using `SignalAggregator` and `MLStrategy`.

```python
from src.crypto_analysis.signals.aggregator import SignalAggregator, WeightedAggregator
from src.crypto_analysis.signals.strategy import MLStrategy

aggregator = WeightedAggregator(generators=[gen1, gen2], weights=[0.6, 0.4])
strategy = MLStrategy(symbols=["BTCUSDT", "ETHUSDT"], aggregator=aggregator)
```

## Backtesting

Use `DataHandler` and `PortfolioManager` to simulate trading.

```python
from src.crypto_analysis.signals.strategy import DataHandler, PortfolioManager

data_handler = DataHandler()
portfolio = PortfolioManager(initial_equity=10000.0)

# Load data and run strategy
data_handler.load_data("BTCUSDT", df)
orders = strategy.generate_signals(data_handler, portfolio)

for order in orders:
    portfolio.execute_order(order, data_handler)
```

## Online Learning

This project supports adaptive online learning models that update in real-time.

- `OnlineLSTM`: LSTM model that can be trained incrementally.
- `OnlineRF`: Random Forest model for online classification.
- `AdaptiveLR`: Adjusts learning rate based on market regime detection.

See `src.crypto_analysis.online` for implementation details.
